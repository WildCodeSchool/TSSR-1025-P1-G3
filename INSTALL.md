# 3. Installation sur le client
<span id="installation-sur-le-client"></span>
    Logiciel necessaire sur le client UBU01 :
        - KeePass2

    Installation de KeePass2 via terminal :
        
        sudo add-apt-repository ppa:ubuntuhandbook1/keepass2
        

        sudo apt update && sudo apt upgrade -y


    Logiciel necessaire sur le client WIN01 :
        - KeePass2

    Installation de KeePass2 :

        - Télécharger l'éxécutable suivant :
            https://sourceforge.net/projects/keepass/
        - Exécuter l'installation en gardant les choix pré-selectionnés
